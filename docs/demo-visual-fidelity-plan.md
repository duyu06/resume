# 子项目视觉一致性与双端测试计划

分支：`feature/demo-visual-fidelity`

## 目标

将四个核心项目的主站封面图替换为根据项目内容生成的视觉稿，并将对应 Demo 页面改造成与视觉稿一致、核心流程可交互、桌面与移动端可稳定使用的产品原型。

核心项目顺序：

1. YOLA 跨境电商独立站
2. 数字人风格对话模型微调
3. 多账号运营 RPA 自动化系统
4. AI 电商主图生成器

## MotionSites 设计机制

从 `xianxian-sensen/motionsites-prompts` 中组合以下机制，而不是复制品牌或素材：

- Interactive Discovery：分层揭示、受控聚光、指针与触摸降级
- Prisma Creative Studio：编辑式排版、电影感媒体、分阶段入场
- Bold Studio：大字号标题、响应式导航、强信息层级
- 3D Jack Portfolio：项目证据、设备展示、克制卡片堆叠

## 项目验收

### YOLA

视觉：与生成图一致的黑色珠宝品牌、戒指主视觉、桌面与移动设备展示、黑白商品陈列和暗红强调色。

交互：

- 首屏视频或视觉帧随滚动平滑变化
- 商品横向展示
- 商品快速查看
- 规格选择
- 加入购物袋及数量反馈
- 购物袋抽屉
- 菜单、订阅和返回作品集

动画性能：

- 使用 GSAP ScrollTrigger 与 requestAnimationFrame
- 视频 currentTime 更新前检查 `video.seeking`
- 揭示动画使用 transform，不使用持续 width 布局重排
- 离开视口后停止视频解码更新
- 移动端独立参数与 reduced-motion 降级

### 数字人模型微调

视觉：深色 AI 工作台、薄荷绿强调、数据构建、Prompt 协议、训练、评分卡、数字人聊天和 API 交付。

交互：

- 数据集筛选与统计变化
- Prompt 风格标签切换
- 训练任务启动、暂停和进度
- 评测维度与版本切换
- 聊天消息发送与模拟回复
- API 示例复制反馈

### RPA 自动化

视觉：浅色 SaaS 控制中心、黄色强调、任务队列、浏览器任务、账号环境、人工审批与日志中心。

交互：

- 任务筛选
- 启动、暂停和重试任务
- 环境组切换
- 人工审批通过或驳回
- 日志状态同步
- 异常状态与恢复流程

### AI 电商主图生成器

视觉：暖珊瑚色生成工作台、商品上传、主体锁定、风格场景、批量生成、对比、评分和导出。

交互：

- 上传示例商品
- 主体锁定开关
- 风格和尺寸选择
- 模拟生成进度
- 生成结果选择
- 原图与生成图对比滑杆
- 质量评分变化
- 单张与批量导出反馈

## 双端测试矩阵

桌面：1024 × 768、1440 × 900、1920 × 1080。

移动：360 × 800、390 × 844、430 × 932，并额外检查横屏。

每个页面检查：

- 无横向溢出
- 导航和弹窗安全区
- 触摸目标至少 44px
- 核心流程可以完整操作
- 加载、空、成功和异常状态
- 键盘、Esc、焦点与 aria-live
- reduced-motion
- GitHub Pages `/resume/` 路径
- JavaScript 语法、TypeScript 和 Vite 构建

## 合并规则

只有在四个核心 Demo 完成核心交互、构建通过，并完成桌面与移动端视觉检查后才将 PR 合并到 `main`。
