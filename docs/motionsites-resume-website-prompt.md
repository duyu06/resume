## 6. 视觉系统

保留当前浅色基调，并将其整理为更加统一的技术编辑风格。

### 6.1 色彩方向

- 页面主背景：近白或极浅冷灰
- 次级背景：浅灰蓝
- 主要文字：深石墨或近黑
- 次级文字：中性灰
- 主强调色：冷青 / 冷蓝
- 强调色只用于关键按钮、连接线、状态和数据
- 项目详情弹窗允许使用更深的背景增强沉浸感

不要把全站改为纯黑背景，也不要使用大面积廉价蓝紫渐变。

### 6.2 字体方向

- 中文正文：现有无衬线字体体系优先
- 英文、数字与 UI：Inter 或当前字体
- 大标题：使用现有 display font
- 技术标签：等宽字体
- 不额外引入不稳定或来源不明的商业字体

### 6.3 纹理

允许：

- 极低透明度 SVG noise
- 细网格
- 轻微径向渐变
- 局部磨砂玻璃
- 细描边

禁止：

- 大量发光粒子
- 高频闪烁
- 过度玻璃拟态
- 复杂 3D 场景
- 影响文字阅读的动态背景
